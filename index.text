Introducción:
-------------

[OpenERP][openerp] es framework completo escrito en python, para soluciones de Gestión Empresarial liberado bajo la licencia [AGPL][agpl]. Basado en [OpenObject][object] y estructurado de modo modular, escalable y [RAD][rad]. 

En este entrenamiento se adiestra sobre el uso de las herramientas que ofrece OpenObject para construir aplicaciones rapidamente integrando soporte de Mapeo de Objeto-Relación ([ORM][orm]), interfaces basadas en planilla llamado "Controlador de Vista del Modelo" ([MVC][mvc]) sistema de generación de reportes, internacionalización automatizada y mucho más.

Orientado para personas con conocimiento en Python que trabajen en el area de las soluciones empresariales para el sector privado o gobierno.
